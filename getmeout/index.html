Get Me Out!

A client-side utility mod for instant game disconnection.

Modrinth Page GitHub Source

Quick Disconnect

Allows players to bypass the standard escape menu. By pressing the . key (configurable), you can instantly disconnect from a server or exit a local world.

Save Notifications

Includes built-in support for toast notifications. These alerts notify the user exactly when world data has been successfully saved to the disk.

Version Support

Compatible with Fabric and Quilt loaders. Actively maintained for Minecraft versions 1.21.x, with developing legacy support backtracking to 1.14.

Technical Specifications

Get Me Out! is designed to be lightweight and focused on a single utility. It requires the Fabric API and Fabric Language Kotlin for 1.21.x environments.
